setiview

setiview is a simple frontend for the unix seti@home client.

I wrote this program in 1999, and I don't support it (or even use it) anymore. I have no idea if it even still works with recent SETI@Home clients.

Usage

Run setiview with the commandline for setiathome as arguments. I normally run it like so:
setiview ./setiathome -nice 20
Note that your setiathome client has to be set up such that you don't have to type anything to it to login (i.e. run it once standalone before using setiview).
